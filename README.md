# Checkout-Felder — Checkout Field Editor for WooCommerce

Edit, reorder, require, hide, and **add** fields to the WooCommerce checkout — grouped by
billing, shipping, and additional. Custom field values are saved to the order and shown in
the admin order screen and the order emails.

## ▶ Live demo (no install)

**[Open the demo in WordPress Playground](https://playground.wordpress.net/?blueprint-url=https://raw.githubusercontent.com/kipphard/checkout-felder/main/demo/blueprint.json)**
— the field editor with a custom "gift message" field added, in your browser.

## Install

Requires **WooCommerce**. Download the latest
[release](https://github.com/kipphard/checkout-felder/releases) zip → WordPress admin →
*Plugins → Add New → Upload Plugin → Activate*. Manage fields under the **Checkout-Felder** menu.

## Features (free)

- Edit existing checkout fields (label, placeholder, required, order) and **hide** unneeded ones
- **Add custom** text + textarea fields
- Field values saved to the order and shown in admin + order emails
- Bilingual (German + English)
